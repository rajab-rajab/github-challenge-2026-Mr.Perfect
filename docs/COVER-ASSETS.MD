# Cover Assets Guide

This document lists the generated branding images for MR. PERFECT OS.

## Available Images

### 1. `project-cover.png`
Primary cinematic cover image.

Recommended for:
- project showcase
- GitHub/social preview image
- landing page hero banner
- presentation title slide

### 2. `project-cover-corporate.png`
Cleaner, more corporate software-branding version.

Recommended for:
- product sheet
- portfolio presentation
- documentation front page
- investor/client style decks

### 3. `project-cover-portrait.png`
Portrait-oriented version designed for documentation-style layouts.

Recommended for:
- printed documentation cover
- PDF manual cover
- report front page
- A4 portrait title page

## Suggested Usage
- Use `project-cover.png` as the default project hero image.
- Use `project-cover-corporate.png` when a more polished enterprise style is preferred.
- Use `project-cover-portrait.png` for document covers and formal project documentation.
